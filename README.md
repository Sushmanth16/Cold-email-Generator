# Cold-email-Generator
AI-powered application that extracts job descriptions and generates personalized outreach emails using Python and Streamlit

## Overview
This project is a lightweight AI-inspired application that generates personalized outreach emails from job descriptions. It extracts key skills from a job post, maps them to relevant portfolio links, and generates a context-aware cold email.

## Tech Stack
- Python
- Streamlit

## Features
- Paste a job description
- Extract relevant technical skills
- Retrieve matching portfolio links
- Generate a personalized email draft

## Project Workflow
1. User pastes a job description
2. App extracts matching technical skills
3. Relevant portfolio links are selected
4. A draft outreach email is generated

pip install -r requirements.txt
